# Percent Manipulative

This project aims to show how to calculate percentages as a multiple of a one percent part.

Example:

```
15% of 150:
1% of 150 = 1.5
15 * 1.5 = 22.5
```

## Prerequisites

- [React](https://reactjs.org/)

## Usage

to run in debug mode:

```
npm start
```

to create a production-ready build:

```
npm run build
```
